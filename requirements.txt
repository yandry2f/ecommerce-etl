# =============================================================================
# NOTAS DE INSTALACIÓN
# =============================================================================
#
# 1. DRIVERS DE BASES DE DATOS:
#    - Para SQL Server en Linux: Instalar Microsoft ODBC Driver
#      https://docs.microsoft.com/en-us/sql/connect/odbc/linux-mac/installing-the-microsoft-odbc-driver-for-sql-server
#    
#    - Para PostgreSQL: psycopg2-binary incluye librerías necesarias
#
# 2. DEPENDENCIAS DEL SISTEMA:
#    Ubuntu/Debian:
#      sudo apt-get update
#      sudo apt-get install python3-dev build-essential
#      sudo apt-get install unixodbc-dev
#    
#    CentOS/RHEL:
#      sudo yum install python3-devel gcc
#      sudo yum install unixODBC-devel
#
# 3. PROBLEMAS COMUNES:
#    - Error con psycopg2: Usar psycopg2-binary en su lugar
#    - Error con pyodbc: Verificar instalación de ODBC drivers
#    - Error con openpyxl: pip install --upgrade setuptools
#
# 4. VERSIONES MÍNIMAS:
#    - Python 3.8+ (recomendado 3.9+)
#    - pip 21.0+
#    - setuptools 50.0+
#
# 5. ENTORNOS ESPECIALIZADOS:
#    Para producción, considere:
#    - Usar poetry o pipenv para gestión de dependencias
#    - Fixar versiones exactas para reproducibilidad
#    - Crear requirements-dev.txt separado para desarrollo
